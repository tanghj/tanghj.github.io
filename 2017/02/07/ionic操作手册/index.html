ionic操作手册

  1. 新建应用

    • 空白应用（ionic start myApp blank）
    • Tabs应用（ionic start myApp tabs）
    • 滑动菜单应用（ionic start myApp sidemenu）

  2. 打包成 ios 或者 Android
    ###iOS

    • $ ionic platform add ios
      • $ ionic build ios
      • $ ionic emulate ios

    ###Android

    • $ ionic platform add android
      • $ ionic build android
      • $ ionic emulate android
